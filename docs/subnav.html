Subnav

Defines different styles for a sub navigation.

Usage

To apply this component, add the .uk-subnav class to an <ul> element and nest <a> elements within the list. Use a <span> element instead to disable the item.

Example

  • Item
  • Item
  • Disabled
  • Item

Markup

<ul class="uk-subnav">
    <li><a href="">...</a></li>
    <li><a href="">...</a></li>
    <li><span>...</span></li>
    <li><a href="">...</a></li>
</ul>

Modifiers

Subnav line

Add the .uk-subnav-line class to separate menu items with lines.

Example

  • Item
  • Item
  • Disabled
  • Item

Markup

<ul class="uk-subnav uk-subnav-line">
    <li>...</li>
</ul>

Subnav pill

Add the .uk-subnav-pill class to highlight the active menu item with a background. To apply an active state to a menu item, just add the .uk-active class.

Example

  • Active
  • Item
  • Disabled
  • Item

Markup

<ul class="uk-subnav uk-subnav-pill">
    <li class="uk-active">...</li>
</ul>

Subnav with dropdowns

This is an example of how to use a subnav with the Dropdown component.

Example

  • Active
  • Item
  • Disabled
  • More
    • Item
    • Another item
    • Header
    • Item
    • Another item
    • Separated item

Markup

<ul class="uk-subnav">
    <li><a href="">...</a></li>

    <!-- This is the container enabling the JavaScript -->
    <li data-uk-dropdown="{mode:'click'}">

        <!-- This is the nav item toggling the dropdown -->
        <a href="">...</a>

        <!-- This is the dropdown -->
        <div class="uk-dropdown uk-dropdown-small">
            <ul class="uk-nav uk-nav-dropdown">
                <li><a href="">...</a></li>
            </ul>
        </div>
    </li>
</ul>
